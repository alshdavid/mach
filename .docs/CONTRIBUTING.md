# Contributing

This project welcomes contributions, feel free to raise a PR.

## Dependencies

- Node.js 18+
- PNPM
- Rust

## Building

```bash
pnpm run build
pnpm run build --release
```

## Running Locally

```bash
# Will build /testing/fixtures/simple
pnpm run build-fixture simple
```

## Testing

```bash
pnpm run build
```
